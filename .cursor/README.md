# Carpeta `.cursor`

Aquí vive la configuración de Cursor para este repo. La normativa del producto sigue en `docs/`; esto es operativo para el equipo y los agentes.

## Contenido

| Carpeta / archivo | Para qué sirve |
|---|---|
| `rules/*.mdc` | Reglas que Cursor aplica según contexto. Apuntan a `docs/` para el detalle. |
| `plans/` | Planes guardados desde Plan Mode (historial, retomar trabajo, contexto para otra sesión). |
| `commands/*.md` | Comandos con `/` en el chat: flujos repetibles sin reescribir el mismo prompt. |

Si en el futuro se usa MCP u hooks, sus archivos pueden vivir aquí también, como en otros repos del equipo.
