# Planes guardados

Desde Plan Mode en Cursor, se puede guardar el plan en el workspace. Conviene hacerlo bajo esta carpeta para que quede versionado en git.

Sirve para:

- documentar cómo se abordó un feature o un refactor
- retomar trabajo interrumpido
- dar contexto a otra persona o a otra sesión del agente

Los archivos son Markdown normales; no sustituyen `docs/STANDARDS.md` ni `docs/ARCHITECTURE.md`.
