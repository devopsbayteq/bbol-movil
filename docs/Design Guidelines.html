Typography

Descargar Lexend

Estilos

Lexend es la tipografía oficial del Banco Bolivariano. Descárgala e instálala antes de aplicar los estilos tipográficos. Según el contexto, elige el estilo que mejor se adapte.

  • Display: para títulos destacados, como los que se muestran en una landing page.
  • Headline: ideal para títulos de secciones o categorías.
  • Body: destinado a los textos principales de la interfaz. El estilo base más usado es Body 2 para el contenido general.
  • Caption: para textos de menor relevancia, como notas legales o mensajes secundarios.

Icons

Ir a FontAwesome

Estilos

Banco Bolivariano utiliza la librería de iconos FontAwesome 6. El estilo Brand se recomienda para elementos accionables y el Neutral para acompañar textos o dar funcionalidad sin resaltar demasiado. Si un icono no es reconocible o no sigue un estándar, acompáñalo con una etiqueta de texto.

  • Classic se usa en elementos accionables, ya que su diseño claro facilita la identificación rápida de acciones.
  • Duotone está pensado para elementos decorativos, aportando un estilo expresivo sin afectar la funcionalidad principal.
