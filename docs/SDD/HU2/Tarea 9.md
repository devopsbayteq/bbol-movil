# 9. Backend: pruebas unitarias de servicio de cuentas propias

**Prioridad**: Media-Alta

**Historia padre**: #29223

## Descripción
Crear pruebas unitarias para el servicio de cuentas propias: mapeo correcto de datos, manejo de respuesta vacía, manejo de errores del core/adapter, y validación de campos obligatorios (tipo, número, saldo). Mockear dependencias (core banking/adapter) y validar códigos de error normalizados.
