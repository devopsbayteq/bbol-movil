# 7. Pruebas unitarias frontend: utilitario de moneda y validación de monto

**Prioridad**: Media-Alta

**Historia padre**: #29223

## Descripción
Crear tests unitarios para: formateo monetario (miles, decimales, signo $), conversión/parsing de monto, validación de &gt;0, 2 decimales, y comparación contra saldo disponible. Cubrir casos borde: 0, 0.001, 1000, 1000000.5, valores con coma/punto ingresados por el usuario.
