# 1. Crear layout de pantalla 'Listado de cuentas propias' (Paso 2)

**Prioridad**: Alta

**Historia padre**: #29223

## Descripción
Implementar la vista de listado de cuentas propias para selección de cuenta destino. Incluir: item con tipo de cuenta + descripción, número de cuenta en claro y saldo disponible. Definir estructura de componentes reutilizables (Card/Row) para cada cuenta y contenedor con scroll. Considerar estados: cargando (skeleton), lista vacía (mensaje) y error (retry).
