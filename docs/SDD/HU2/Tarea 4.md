# 4. Maquetar pantalla 'Ingreso del monto' con secciones Origen/Destino

**Prioridad**: Alta

**Historia padre**: #29223

## Descripción
Construir UI con dos secciones: (1) 'Cuenta Origen' con título, tipo+numero en claro y saldo disponible; (2) cuenta destino/beneficiaria con nombre del titular, tipo+descripción, número en claro y saldo disponible. Incluir campos de entrada: monto y motivo, y un botón de acción (continuar). Definir componentes para encabezados de cuenta reutilizables.
