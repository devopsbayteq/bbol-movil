# 1. Maquetar pantalla Paso 3 Confirmación de transferencia propia

**Prioridad**: Alta

**Historia padre**: #29253

## Descripción
Implementar la UI del paso 3 (Confirmación) siguiendo el layout definido: secciones 'Cuenta de origen', 'Cuenta de destino', 'Detalle de la transferencia' y 'Leyenda'. Incluir botón primario  siempre habilitado. Usar componentes reutilizables (cards/rows) y estilos consistentes con el flujo anterior (tipografías, espaciados). Dejar la pantalla conectada a un ViewModel/Presenter/Controller para recibir un modelo de datos (sin lógica de negocio en la vista).
