# 17. Backend: Implementar servicios para detalle/confirmación de transferencia (general)

**Prioridad**: Alta

**Historia padre**: #29253

## Descripción
Crear/ajustar endpoint(s) que provean la data requerida para el comprobante: monto, datos de cuenta débito/crédito (titular, tipo, banco, cuenta enmascarada o token para enmascarar en front), fecha/hora de transacción y número de referencia (comprobante), además del valor de &quot;Servicios financieros&quot; si aplica. Definir contrato estable (versionado si corresponde) y manejo de errores/timeout.
