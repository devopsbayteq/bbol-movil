# 8. Agregar pruebas unitarias

**Prioridad**: Media-Alta

**Historia padre**: #29253

## Descripción
Crear pruebas unitarias para: (1) construcción del UI model (origen/destino/montos/leyenda), (2) enmascarado de cuenta destino, (3) llamado al servicio al presionar TRANSFERIR, (4) control anti-doble envío, (5) navegación a éxito en respuesta OK, (6) manejo de errores (mapeo a mensaje). Usar mocks del servicio y scheduler/dispatcher de pruebas según plataforma.
