# 12. Implementar enmascaramiento de número de cuenta (últimos 3 dígitos)

**Prioridad**: Alta

**Historia padre**: #29253

## Descripción
Crear helper/utilidad compartida para formatear números de cuenta mostrando solo los últimos 3 dígitos (ej: *********123) y reutilizarla en las secciones &quot;Para&quot; y &quot;De&quot;. Incluir manejo de nulos/longitudes menores a 3 y pruebas de formato para no exponer datos sensibles en UI/logs.
