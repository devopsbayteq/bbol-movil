# 10. Implementar manejo de errores y mensajes de UX para Login/OTP

**Prioridad**: Media-Alta

**Historia padre**: #29065

## Descripción
Definir y aplicar catálogo de mensajes para errores comunes (validación, credenciales inválidas, problemas de red, OTP inválido/expirado, demasiados intentos). Asegurar que no se revelen detalles sensibles. Mostrar feedback consistente (toast/dialog/inline) según guía UI.
