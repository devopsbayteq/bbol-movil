# 8. Implementar almacenamiento seguro para tokens/credenciales biométricas

**Prioridad**: Alta

**Historia padre**: #29065

## Descripción
Configurar almacenamiento seguro (Keychain/Keystore/Secure Storage) para persistir tokens y/o secreto necesario para login biométrico. Asegurar cifrado, control de acceso ligado a biometría si la plataforma lo permite, y limpieza al logout o al detectar invalidación. No almacenar contraseña en texto plano.
