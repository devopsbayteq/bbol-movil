# Creación de pantalla de login

## Descripción
<div><p style="margin:0cm 0cm 8pt;font-size:11pt;font-family:Aptos, sans-serif;"><b><span lang=ES-EC style="font-size:12.0pt;line-height:107%;">Como</span></b><span lang=ES-EC style="font-size:12.0pt;">
cliente de los canales digitales <b>necesito</b> disponer de una pantalla que
me permita iniciar sesión en la app <b>para</b> </span><span lang=ES-EC>tener
una experiencia de navegación más simple, intuitiva y funcional.</span> </p> </div>

## Criterios de aceptación
<p style="margin:0cm 0cm 8pt;font-size:11pt;font-family:Aptos, sans-serif;"><span lang=ES-EC style="font-size:12.0pt;">Para
que la historia se de por aceptada debe cumplir con los siguientes puntos:</span> </p><p style="margin:0cm 0cm 8pt;font-size:11pt;font-family:Aptos, sans-serif;"><b><u><span lang=ES-EC style="font-size:12.0pt;">Escenario: Pantalla de Inicio de sesión</span></u></b> </p><ul style="margin-bottom:0cm;"><li><span lang=ES-EC style="font-size:12.0pt;">Al descargar la aplicación, se debe visualizar la pantalla de
Inicio de sesión</span> </li><li><span lang=ES-EC style="font-size:12.0pt;">Esta pantalla dispondrá de los siguientes elementos:</span> </li><ul style="margin-bottom:0cm;"><li><span lang=ES-EC style="font-size:12.0pt;">Saludo inicial</span> </li><li><span lang=ES-EC style="font-size:12.0pt;">Campo de Alias o usuario</span> </li><li><span lang=ES-EC style="font-size:12.0pt;">Campo de contraseña</span> </li><li><span lang=ES-EC style="font-size:12.0pt;">Botón de Inicio de sesión</span> </li><li><span lang=ES-EC style="font-size:12.0pt;">Botón de Crear usuario - Implementar</span> </li><li><span lang=ES-EC style="font-size:12.0pt;">Botón de Solicitar
Productos - Implementar - Cta ahorro / Seguros /&nbsp;</span> </li><li><span lang=ES-EC style="font-size:12.0pt;">Botón de Contactos - Implementar a que medios se puede contactar el cliente.</span> </li> </ul><ul style="margin-bottom:0cm;"><li><span lang=ES-EC style="font-size:12.0pt;">Versión de la app -&nbsp;</span> </li> </ul> </ul><p style="margin:0cm 0cm 8pt;font-size:11pt;font-family:Aptos, sans-serif;"><span lang=ES-EC style="font-size:12.0pt;">&nbsp;</span> </p><p style="margin:0cm 0cm 8pt;font-size:11pt;"><span lang=ES-EC style="font-size:12pt;"><font face="Aptos, sans-serif">Escenario:&nbsp;</font></span> </p><ul style="box-sizing:border-box;padding:0px 0px 0px 40px;font-size:14px;margin-bottom:0cm;"><li style="box-sizing:border-box;"><span lang=ES-EC style="box-sizing:border-box;font-size:12pt;">Campo de Alias o usuario</span> </li><li style="box-sizing:border-box;"><span lang=ES-EC style="box-sizing:border-box;font-size:12pt;">Campo de contraseña</span> </li><li style="box-sizing:border-box;"><span lang=ES-EC style="box-sizing:border-box;font-size:12pt;">Botón de Inicio de sesión</span> </li> </ul><div><br> </div><p> </p><p style="margin:0cm 0cm 8pt;font-size:11pt;font-family:Aptos, sans-serif;"><span lang=ES-EC style="font-size:12.0pt;"><br></span> </p><p style="margin:0cm 0cm 8pt;font-size:11pt;font-family:Aptos, sans-serif;"><b><u><span lang=ES-EC style="font-size:12.0pt;">Escenario: Validaciones en el ingreso de las credenciales</span></u></b> </p><p style="margin:0cm 0cm 8pt;font-size:11pt;font-family:Aptos, sans-serif;"><b><span lang=ES-EC style="font-size:12.0pt;">Usuario o alias</span></b> </p><ul style="margin-bottom:0cm;"><li><span lang=ES-EC style="font-size:12.0pt;">Es un campo alfanumérico, es decir se permite el ingreso de
letras, números.</span> </li><li><span lang=ES-EC style="font-size:12.0pt;">También se permite el ingreso de caracteres especiales como:<b></b></span> </li><ul style="margin-bottom:0cm;"><li><span lang=ES-EC style="font-size:12.0pt;">Punto (.)<b></b></span> </li><li><span lang=ES-EC style="font-size:12.0pt;">Guion ( - )<b></b></span> </li><li><span lang=ES-EC style="font-size:12.0pt;">Subguion ( - )<b></b></span> </li> </ul><li><span lang=ES-EC style="font-size:12.0pt;">Cantidad de caracteres permitidos:<b></b></span> </li><ul style="margin-bottom:0cm;"><li><span lang=ES-EC style="font-size:12.0pt;">Mínimo 12</span> </li><li><span lang=ES-EC style="font-size:12.0pt;">Máximo 16 caracteres. </span> </li> </ul> </ul><p style="margin:0cm 0cm 8pt 36pt;font-size:11pt;font-family:Aptos, sans-serif;margin-left:72.0pt;"><b><span lang=ES-EC style="font-size:12.0pt;">&nbsp;</span></b> </p><p style="margin:0cm 0cm 8pt;font-size:11pt;font-family:Aptos, sans-serif;"><b><span lang=ES-EC style="font-size:12.0pt;">Contraseña:</span></b> </p><ul style="margin-bottom:0cm;"><li><span lang=ES-EC style="font-size:12.0pt;">Tendrá el icono del ojito que permitir ocultar o mostrar la
contraseña, en donde:</span> </li><ul style="margin-bottom:0cm;"><li><span lang=ES-EC style="font-size:12.0pt;">Ojo abierto </span><span lang=ES-EC style="font-size:12.0pt;font-family:Wingdings;"><span>è</span></span><span lang=ES-EC style="font-size:12.0pt;"> Se muestra la contraseña</span> </li><li><span lang=ES-EC style="font-size:12.0pt;">Ojo cerrado </span><span lang=ES-EC style="font-size:12.0pt;font-family:Wingdings;"><span>è</span></span><span lang=ES-EC style="font-size:12.0pt;"> Se oculta la contraseña.</span> </li> </ul><li><span lang=ES-EC style="font-size:12.0pt;">La contraseña debe enmascararse.</span> </li><li><span lang=ES-EC style="font-size:12.0pt;">Se utilizará como enmascaramiento el punto.</span> </li> </ul><p style="margin:0cm 0cm 8pt;font-size:11pt;font-family:Aptos, sans-serif;"><span lang=ES-EC style="font-size:12.0pt;">&nbsp;</span> </p><p style="margin:0cm 0cm 8pt;font-size:11pt;font-family:Aptos, sans-serif;"><b><u><span lang=ES-EC style="font-size:12.0pt;">Escenario: Botón de Inicio de sesión</span></u></b> </p><ul style="margin-bottom:0cm;"><li><span lang=ES-EC style="font-size:12.0pt;">Este botón iniciara como deshabilitado.</span> </li><li><span lang=ES-EC style="font-size:12.0pt;">Una vez que se hayan ingresado tanto el usuario y la
contraseña se habilitara el botón.</span> </li><li><span lang=ES-EC style="font-size:12.0pt;">Al dar clic sobre este botón se deberán realizar las
validaciones respectivas.</span> </li> </ul><p style="margin:0cm 0cm 8pt;font-size:11pt;font-family:Aptos, sans-serif;"><b><u><span lang=ES-EC style="font-size:12.0pt;"></span></u></b> </p><p style="margin:0cm 0cm 8pt;font-size:11pt;font-family:Aptos, sans-serif;"><b><u><span lang=ES-EC style="font-size:12.0pt;">Escenario: validación de Credenciales</span></u></b> </p><ul style="margin-bottom:0cm;"><li><span lang=ES-EC style="font-size:12.0pt;">Cuando las credenciales
sean incorrectas se mostrará el mensaje <b><i>“Usuario y/o contraseña
incorrecta”</i></b></span> </li><li><span lang=ES-EC style="font-size:12.0pt;">Caso contrario,
direccionara a la posición consolidada del canal.<b><i></i></b></span> </li> </ul><p style="margin:0cm 0cm 8pt;font-size:11pt;font-family:Aptos, sans-serif;"><b><u><span lang=ES-EC style="font-size:12.0pt;"></span></u></b> </p><p style="margin:0cm 0cm 8pt;font-size:11pt;font-family:Aptos, sans-serif;"><b><u><span lang=ES-EC style="font-size:12.0pt;">Escenario: Compatibilidad entre plataformas.</span></u></b> </p><ul style="margin-bottom:0cm;"><li><span lang=ES-EC style="font-size:12.0pt;">Esta funcionalidad debe ser la misma en ambas plataformas.</span> </li><ul style="margin-bottom:0cm;"><li><span lang=ES-EC style="font-size:12.0pt;">Validar en dispositivo IOS</span> </li><li><span lang=ES-EC style="font-size:12.0pt;">Validar en dispositivo
