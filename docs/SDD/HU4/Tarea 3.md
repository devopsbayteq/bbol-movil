# 3. Implementar validaciones de campo Contraseña (6-15, mayúscula/número/especial)

**Prioridad**: Alta

**Historia padre**: #29065

## Descripción
Agregar validación del input &quot;Contraseña&quot;: longitud mínima 6 y máxima 15; requerir al menos 1 letra mayúscula, 1 número y 1 caracter especial. Mantener input enmascarado (password). Mostrar mensajes de error y estado inválido conforme a componentes del front.
