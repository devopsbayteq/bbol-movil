# 4. Agregar pruebas unitarias front de componentes y validaciones (monto/motivo/botón)

**Prioridad**: Media-Alta

**Historia padre**: #29251

## Descripción
Crear tests unitarios para: (1) formateo de monto (miles/decimales, límite 2 decimales, entradas parciales), (2) validación de habilitación del botón , (3) validación/sanitización de motivo (máx 30, caracteres permitidos), (4) snapshot o rendering básico de AccountCard. Usar el framework del proyecto (Jest + Testing Library / equivalente).
