# 1. Frontend: Crear pantalla inicial de Transferencias con tipos y favoritos

**Prioridad**: Alta

**Historia padre**: #29067

## Descripción
Construir pantalla inicial del módulo con opciones: 'Entre mis cuentas', 'Terceros (Bolivariano)', 'Otros bancos' y sección 'Contactos favoritos'.
